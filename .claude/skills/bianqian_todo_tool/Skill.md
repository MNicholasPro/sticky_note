# Skill Content

## 概述
本 Skill 将指导你构建一个跨平台的桌面便签应用。该应用结合了 Electron 的强大功能与 Web 前端技术的灵活性。主要特性包括动态主题背景、交互音效、基于颜色的状态分类（待做/再做/完成），以及完整的 Mac 系统托盘集成（支持隐藏/显示窗口、显示待办统计）。

## 功能如下：
- 界面需要有丰富多彩的背景，可选择多种主题背景或随机生成背景。
- 添加各种点击按钮的音效。
- 点击添加事件按钮，弹出添加事件窗口。
- 每个事件记录包含多个信息：创建时间、更新时间、事件名称、事件内容、事件等级、事件状态、事件操作（支持添加事件、删除事件、标记事件、编辑事件，标记事件支持下拉选择待做、再做、完成状态）
- 事件等级：高、中、低，使用不同颜色区分，高使用红色背景色，中使用蓝色背景色，低使用灰色背景色。
- 标记事件支持3种状态：待做、再做、完成，待做使用黄色背景色，每个事件只能支持一种状态，再做使用橙色背景色，完成使用绿色背景色，新增事件则默认无背景色。
- 支持打包成Mac应用，状态栏展示便签图标，点击状态栏图标支持隐藏/显示便签窗口页面。
- 点击状态图标需要展示不同状态下事件数量。
- 点击关闭后支持后台运行。
- 支持持久化保存事件。

## 前置条件
1.  已安装 [Node.js](https://nodejs.org/) (推荐 v14 或更高版本)。
2.  已安装代码编辑器（如 VS Code）。

## 安装与初始化

### 1. 创建项目目录并初始化
打开终端，执行以下命令：

```bash
npm init -y
```

### 2. 安装依赖
安装 Electron 核心包及打包工具：

```bash
npm install electron electron-builder --save-dev
```

### 3. 项目结构规划
建议创建以下文件结构：
```text
my-sticky-notes/
├── main.js            # 主进程逻辑
├── preload.js         # 预加载脚本 (安全桥接)
├── index.html         # 主页面结构
├── style.css          # 样式与主题
├── app.js             # 渲染进程逻辑
├── sounds/            # 音效文件目录 (可选，代码中提供播放逻辑)
└── package.json       # 项目配置
```

---

## 构建与运行

### 代码构建
1.  按照项目结构规划创建文件。
2.  按照功能要求逻辑，编写各个文件的内容。
3.  将代码写入对应文件中，确保主进程、渲染进程和样式逻辑正确连接。

### 运行应用
在终端中执行：
```bash
npm start
```

### 打包应用 (Mac DMG)
安装打包依赖（通常已安装）并执行：
```bash
npm run build
```
生成的 `.dmg` 文件将位于 `dist` 目录中。

## 总结
通过上述代码，我们构建了一个功能完备的桌面便签应用。
*   **持久化**: 使用 IPC 在主进程读写 JSON 文件。
*   **状态管理**: 通过 `data-status` 属性在 CSS 中渲染不同颜色的边框和背景。
*   **系统集成**: 实现了点击 Dock 图标或系统托盘图标的切换逻辑，以及关闭窗口隐藏到后台的逻辑。
*   **音效**: 使用 Web Audio API 生成简单的提示音，无需额外资源文件。
