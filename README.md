# 📝 智能便签应用 (Sticky Notes App)

这是一个基于 **Electron** 开发的高效、美观的桌面便签管理工具。它支持自定义主题、状态管理、以及深度集成 macOS/Windows 的系统托盘功能，旨在为用户提供一个轻量级的灵感记录与任务追踪空间。

## ✨ 核心功能

### 1. 灵活的便签管理
- **快速创建**：一键生成新便签。
- **状态追踪**：支持将便签标记为不同的状态（待办、进行中、已完成）。
- **优先级分级**：通过颜色区分优先级（高、中、低），直观掌握任务紧迫程度。
- **持久化存储**：所有数据本地存储，无需担心隐私泄露，重启应用后数据自动恢复。

### 2. 深度系统集成
- **系统托盘 (System Tray)**：应用运行在后台，可通过托盘图标快速唤起或退出。
- **托盘菜单**：直接从托盘菜单启动新便签或查看统计。
- **后台运行**：关闭主窗口后，应用继续在后台运行，确保不丢失状态。

### 3. 视觉与交互
- **动态主题**：支持多种配色方案，满足不同审美需求。
- **响应式布局**：便签窗口可自由调整大小，布局自动适配。
- **交互动画**：流畅的过渡效果，提升用户体验。

## 🚀 快速开始

### 前置条件
确保你的系统中已安装 [Node.js](https://nodejs.org/) (建议版本 v16+)。

### 安装步骤
1. **克隆项目**
   ```bash
   git clone <your-repo-url>
   cd <project-folder>
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动应用**
   ```bash
   npm start
   ```

### 打包发布
如果你想将其打包为可执行文件 (`.exe` 或 `.app`)：
```bash
npm run build
```

## 📂 项目结构

```text
├── assets/             # 静态资源（图标、图片）
├── src/
│   ├── main/           # 主进程逻辑 (Electron Main Process)
│   │   └── main.js     # 窗口管理、托盘配置、系统交互
│   ├── renderer/       # 渲染进程逻辑 (Frontend)
│   │   ├── index.html  # 主界面结构
│   │   ├── style.css   # 样式定义（含主题颜色）
│   │   └── renderer.js # 业务逻辑与 DOM 操作
│   └── preload.js      # 预加载脚本（桥接主进程与渲染进程）
├── package.json        # 项目配置与依赖
└── README.md           # 项目说明文档
```

## ⚙️ 配置说明

在 `src/main/main.js` 中，你可以自定义以下配置：
- **窗口初始尺寸**：修改 `width` 和 `height`。
- **托盘图标路径**：修改 `Tray` 实例化时的图标路径。
- **默认主题**：在 CSS 中修改 `--primary-color` 等变量。
